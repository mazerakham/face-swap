Base URL: https://api.openai.com/v1

## Image Generation

POST /images/generations
Generate images from a text prompt

Params:
```json
{
  "model": "dall-e-3",
  "prompt": "a white siamese cat",
  "n": 1,
  "size": "1024x1024",
  "quality": "standard"
}
```

cURL Request:
```bash
curl https://api.openai.com/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -d '{
    "model": "dall-e-3",
    "prompt": "a white siamese cat",
    "n": 1,
    "size": "1024x1024",
    "quality": "standard"
  }'
```

Response:
```json
{
  "created": 1589478378,
  "data": [
    {
      "url": "https://..."
    }
  ]
}
```

## Image Editing

POST /images/edits
Create an edited or extended image given an original image and a prompt

Params:
- image: The image to edit. Must be a valid PNG file, less than 4MB, and square.
- mask: An additional image whose fully transparent areas (e.g. where alpha is zero) indicate where image should be edited. Must be a valid PNG file, less than 4MB, and have the same dimensions as image.
- prompt: A text description of the desired image(s).
- n: The number of images to generate. Must be between 1 and 10.
- size: The size of the generated images. Must be one of 256x256, 512x512, or 1024x1024.

cURL Request:
```bash
curl https://api.openai.com/v1/images/edits \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -F image="@original_image.png" \
  -F mask="@mask.png" \
  -F prompt="A sunlit indoor lounge area with a pool containing a flamingo" \
  -F n=1 \
  -F size="1024x1024"
```

Response:
```json
{
  "created": 1589478378,
  "data": [
    {
      "url": "https://..."
    }
  ]
}
```
